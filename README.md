# SETHU FOOTBALL CLUB - Billing Software

A professional, streamlined receipt generation application built for the **Madurai District Football Association** and **Sethu Football Club**. This software is optimized for high-speed offline billing and professional documentation.

## 🚀 Key Features
- **Instant Receipt Generation:** Clean, standardized receipt templates.
- **Auto-Calculated Amounts:** Automatically converts numeric currency values into Indian English words (Lakhs, Crores, etc.).
- **Professional Print Layout:** Optimized for **A4 Landscape** printing with support for **2 Copies per sheet**.
- **Local Persistence:** All entries are auto-saved to your local device memory as you type.
- **Dual Platform Support:** Runs as a native **Windows Executable (.exe)** and as a **Global Web Application**.

## 💻 Desktop Usage (Windows)
1.  **Direct App:** Double-click `SETHU FC BILLING.exe` for a silent, distraction-free app experience.
2.  **Classic Mode:** Run `SETHU FC BILLING.bat` to launch via your default browser.
3.  **Shortcut Creator:** Use `Create_Desktop_Shortcut.ps1` to pin a professional branded icon to your Desktop.

## 🌐 Web Usage
This project is configured for one-click hosting on **GitHub Pages**. Access your billing records from any mobile device or tablet while on the field.

## 🛠️ Code Architecture
- `index.html`: Main UI Structure.
- `css/style.css`: Print-optimized layout and branding.
- `js/script.js`: Core logic for word calculation, storage, and printing.
- `assets/images/`: Logos, Seals, and Signatures.
